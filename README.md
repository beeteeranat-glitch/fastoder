# FastOrder

เว็บสั่งเครื่องดื่ม/อาหารออนไลน์ พร้อมระบบติดตามออเดอร์แบบ Realtime, ระบบสะสมแต้ม และระบบแลกเครื่องดื่มฟรี สำหรับร้านที่รองรับการชำระเงินด้วย **เงินสด** และ **โอนเงินพร้อมอัปโหลดสลิป**

---

# สถานะโปรเจกต์

✅ พร้อมใช้งานสำหรับร้านค้าที่รองรับ

* เงินสด
* โอนเงิน + อัปโหลดสลิป
* Order Tracking แบบ Realtime
* ระบบสะสมแต้ม
* ระบบแลกเครื่องดื่มฟรี
* รายงานลูกค้าและเมนูขายดี

Migration ที่ต้องรัน

```
009_loyalty_customer_auth.sql
011_customer_address.sql
```

# ฟีเจอร์ที่ทำแล้ว

## ระบบลูกค้า

* ✅ สั่งซื้อได้ทันที ไม่ต้อง Login (Guest Checkout)
* ✅ กรอกชื่อ เบอร์โทร และที่อยู่ใน Checkout
* ✅ ติดตามออเดอร์ด้วยเลขออเดอร์ (`/orders/[orderNumber]`)
* ✅ หน้าเมนูขายดีบนหน้าแรกและ `/menu`

> หมายเหตุ
>
> โปรเจกต์เลือกใช้ **Guest Checkout** เป็นหลัก และสามารถใช้งานระบบคะแนนและการแลกสิทธิ์ผ่านข้อมูลลูกค้าในระบบได้

# ระบบสะสมแต้ม

* ✅ ค่าเริ่มต้น 1 บาท = 1 คะแนน
* ✅ ได้คะแนนเมื่อออเดอร์มีสถานะ `COMPLETED`
* ✅ ตรวจสอบคะแนนได้
* ✅ ดูประวัติการได้รับคะแนน
* ✅ ดูประวัติการแลกคะแนน

> หมายเหตุ
>
> ค่า 1 บาท = 1 คะแนน เป็นค่าเริ่มต้นของโปรเจกต์ และสามารถปรับเปลี่ยนได้ในอนาคต

---

# ระบบแลกเครื่องดื่มฟรี

เงื่อนไข

* ✅ ใช้ 100 คะแนน ต่อ 1 สิทธิ์
* ✅ แลกเครื่องดื่มฟรี 1 แก้ว
* ✅ ครอบคลุมเฉพาะราคาเครื่องดื่มหลัก
* ✅ Topping / Add-on คิดเงินเพิ่มตามปกติ
* ✅ แก้วใหญ่หรือเพิ่มตัวเลือกพิเศษ ชำระเฉพาะส่วนต่าง
* ✅ คะแนนถูกหักทันทีเมื่อยืนยันการแลก

ตัวอย่าง

```
เครื่องดื่ม 40 บาท

ไข่มุก 10 บาท
โอรีโอ 10 บาท

ใช้สิทธิ์ฟรี

ลูกค้าชำระเพิ่ม 20 บาท
```

# หมายเหตุ

* โปรเจกต์ใช้ Guest Checkout เป็นหลัก
* ไม่จำเป็นต้องสมัครสมาชิกก่อนสั่งซื้อ
* ระบบสะสมแต้มอ้างอิงจากหมายเลขโทรศัพท์ลูกค้า
* คะแนนจะถูกเพิ่มเมื่อออเดอร์มีสถานะ `COMPLETED`
* รองรับการอัปเดตสถานะแบบ Realtime ผ่าน Supabase Realtime
* ระบบพิมพ์ใบออเดอร์รองรับ Browser Print แล้ว
