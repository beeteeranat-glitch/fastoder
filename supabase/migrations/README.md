# Migrations

ไฟล์ที่รันบน Supabase แล้วถูกย้ายไป `archive/` (ตรวจ schema บน DB แล้วครบ 001–013)

## โฟลเดอร์นี้
- ใส่ **migration ใหม่** ที่ยังไม่ได้รัน (เช่น `014_xxx.sql`)

## รัน migration
```bash
npm run db:migrate          # รัน archive + ไฟล์ใหม่ในโฟลเดอร์นี้
npm run db:migrate:011      # รันไฟล์เดียว (หาใน archive หรือโฟลเดอร์นี้)
```
